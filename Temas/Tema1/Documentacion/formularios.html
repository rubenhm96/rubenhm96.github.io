Formularios

FORMULARIOS

Los formularios son la herramienta esencial para cualquier aplicación, ya que todas las aplicaciones deben gestionar datos y esos datos deben ser introducidos y manipulados a través de formularios.

TEXT

Es el campo más utilizado y versátil. Se trata de una caja de texto

Ejemplo

TEXTAREA

Se trata de una caja de texto multilínea.

Ejemplo

SEARCH

Campo de texto específico para Búsquedas.

Ejemplo

PASSWORD

Campo muy usado, se trata de una caja de texto cuyos caracteres son enmascarados con asteriscos.

Ejemplo

NUMBER

Cajas de texto en las que solamente pueden introducirse valores numéricos. Muy útiles en dispositivos móviles.

Ejemplo

Rubén Herrera Marcos

2º DAW
