Testear Accesibilidad

Para llevar a cabo el testeo de la accesibilidad de una página Web se pueden realizar las siguientes prácticas:

  • Hacer un testeo de la navegación con teclado de la Web. Para tener una buena experiencia, intentar tener un orden de tabulación lógico.
  • Comprobar que el contenido que está fuera de la pantalla no recibe el foco.
  • Comprobar si la web es Responsive, para que la página Web se puede usar en todo tipo de dispositivos.
  • Probar un lector de pantalla. Comprobar el funcionamiento de lectores de Pantalla como (NVDA o la extensión Screen Reader del navegador Chrome), percatándose de que accedan al contenido que queremos que sea leído.
  • Comprobar si los enlaces y botones indican lo que hacen y su estado.
